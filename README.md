# Dotfiles

This repository contains my personal dotfiles, which are configuration files for various applications and tools that I use on my computer.
These dotfiles help me customize my environment and improve my productivity.

## Contents
- Shell: Configuration for my shell (e.g., bash, zsh, fish, etc.) including aliases, functions, and prompt customization.
  - Currently using `zsh` with `oh-my-zsh`.
- Editor: Configuration for my code editor (e.g., Vim, Emacs, VSCode, etc.) including keybindings, themes, and plugins.
    - Currently using `Neovim` with various plugins for enhanced functionality.
    - This is a submodule, so you will need to initialize and update the submodule after cloning this repository:
    ```bash
    git submodule update --init --recursive
    ```
